# 常用命令
- 新建文件 

    ```
    hexo new fileName
    ```
- 生成静态文件 
    
    ```
    hexo generate || hexo g
    ```
- 启动服务器

    ```
    hexo server  || hexo s
    ```
- 部署网站

    ```
    hexo deploy  || hexo d
    ```
- 清理缓存文件 

    ```
    hexo clean
    ```
# 常用组合
- 启动服务

    ```
    hexo g & hexo s
    ```
- 部署网站

    ```
    hexo clean && hexo deploy
    ```
